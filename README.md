# Quiz Bíblico

Quiz feito para comemorar o dia da bíblia 14/12

Esse quiz contém questões de opções e de verdadeiro ou falso, sobre curiosidades e trechos da bíblia

### Como Jogar

Basta acessar o link do jogo que está aqui a baixo ou ler o qr code 

[link para o quiz](https://guilh3rme2007.github.io/Quiz_biblico/)

![Qr Code para o quiz](images/qrcode_pagina.jpg)


Instruções:
Para começar basta apertar o botão Começar Quiz!
Para selecionar a resposta basta clicar na opção desejada
Depois de escolher a opção correta ficará verde e as demais ficarão vermelhas
O fundo da tela mudará de cor de acordo com a resposta 
Serão sempre dez perguntas e no final pode começar novamente
